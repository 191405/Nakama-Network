## 🎯 Your Mission (If You Choose to Accept)

1. **Step 1:** Run `npm install`

2. **Step 2:** Create `.env` file with Firebase & Gemini keys
   - Copy from `.env.example`
   - Get Firebase config from https://console.firebase.google.com/
   - Get Gemini key from https://makersuite.google.com/app/apikey

3. **Step 3:** Run `npm run dev`

4. **Step 4:** Open http://localhost:5173

5. **Step 5:** Login and explore!
